Requires Python 3.10+. Install (one-time):

pip install ytminer-client

Start downloading:

ytminer-download --server SERVER_URL --output ./videos

Update client:

pip install -U ytminer-client

Google Colab:

!pip install ytminer-client
!ytminer-download --server SERVER_URL --output ./videos --upload

Options:

--worker-name NAME       your name (default: hostname)
--channel @geonews       download specific channel only
--delay 10               seconds between downloads (default: 30)
--batch-size 50          videos per batch (default: 50)
--upload                 upload videos to server after download
--keep-files             keep local files after upload (default: delete)
--cookies FILE           path to cookies.txt
--verbose                show debug logs

For ephemeral environments (Google Colab, etc.), use --upload to stream videos to the server immediately after download. Local files are deleted after upload to save disk.

Browser cookies are auto-detected if YouTube blocks you. Rate limiting, retries, bot cooldown, and error reporting are all handled automatically.
